Making a Web page
The basics of HTML

What is one thing that every website has in common? They all use a language called HTML. HTML is the back-bone of every website you visit.

On this web page, you will learn how to create your very own web page. You will learn the basics of HTML and how to use it.

To make a web page, you must first create a .html file. This file will contain all of your html code.

So what do we name this file? In every directory of your website, you should have an index.html file. When you make a connection to a website, the server will return the default page. This default page is the index.html file.

So name that file index.html!

Now for the code! Open the file up in your favorite text editor. I highly recommend Sublime Text but you can use which ever text editor you are comfortable with.

<html>
    <body>
        hello!
    </body>
</html>
